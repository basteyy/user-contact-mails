User Contact Mails allows users to store additional contact email addresses on their profile page.

== Description ==
With User Contact Mails you can:
* Let users add multiple additional email addresses to their profile.
* Optionally require at least one extra email.
* Let users choose a “primary” address.
* Optionally assign labels to each address (e.g. “Work”, “Home”).
* Administrators can view, filter by role, export, and clear logs of saved emails.
* All settings appear under Tools <span aria-hidden="true" class="wp-exclude-emoji"><span aria-hidden="true" class="wp-exclude-emoji"><span aria-hidden="true" class="wp-exclude-emoji"><span aria-hidden="true" class="wp-exclude-emoji"><span aria-hidden="true" class="wp-exclude-emoji"><span aria-hidden="true" class="wp-exclude-emoji">→</span></span></span></span></span></span> User Contact Mails.

== Installation ==
1. Upload the `user-contact-mails` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the “Plugins” screen in WordPress.
3. Visit **Tools <span aria-hidden="true" class="wp-exclude-emoji"><span aria-hidden="true" class="wp-exclude-emoji"><span aria-hidden="true" class="wp-exclude-emoji"><span aria-hidden="true" class="wp-exclude-emoji"><span aria-hidden="true" class="wp-exclude-emoji"><span aria-hidden="true" class="wp-exclude-emoji">→</span></span></span></span></span></span> User Contact Mails** to configure settings and view saved emails.

== Frequently Asked Questions ==
= How do I reset the log file? =
Go to **Tools <span aria-hidden="true" class="wp-exclude-emoji"><span aria-hidden="true" class="wp-exclude-emoji"><span aria-hidden="true" class="wp-exclude-emoji"><span aria-hidden="true" class="wp-exclude-emoji"><span aria-hidden="true" class="wp-exclude-emoji"><span aria-hidden="true" class="wp-exclude-emoji">→</span></span></span></span></span></span> User Contact Mails**, switch to the “Logs” tab, and click “Clear Log”.

= Can I limit which roles see the extra emails? =
Yes. Under the “Settings” tab you can choose which user roles can view emails and logs.
